# App #028 — Smart Color & Design Tools

Browser-only color and design utility suite for 1 Hub Apps.

Features:
- HEX / RGB / HSL conversion
- Shades and tints
- Image dominant-palette extraction
- Complementary / analogous / triadic harmonies
- WCAG contrast checker
- CSS linear-gradient generator
- Palette CSS export
- Shared ToolAppHeader and LocalProcessingBadge
- Discoverable chat/action metadata in actions.ts

No server upload is required for image processing.
