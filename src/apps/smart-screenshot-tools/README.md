# App #016 — Smart Screenshot Tools

Browser-only screenshot utility for 1 Hub Apps.

Included:
- Multiple screenshot upload and reorder
- Vertical long-screenshot stitching
- Blur sensitive areas
- Rectangle, highlight, arrow and text annotations
- PNG/JPG/WebP export
- Quality, padding, gap and background controls
- Clipboard copy where supported
- Shared ToolAppHeader and LocalProcessingBadge integration
- Chat/action-ready action metadata

Install slug: `smart-screenshot-tools`
Page export: `SmartScreenshotToolsPage`
Route: `/apps/smart-screenshot-tools`
