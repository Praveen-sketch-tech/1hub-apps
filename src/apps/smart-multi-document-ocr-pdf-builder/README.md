# App #033 — Smart Multi-Document OCR & PDF Builder

- Multiple images and PDFs
- Stacked colour previews
- Gentle colour-preserving enhancement
- Official PaddleOCR.js in-browser OCR
- Hindi + English via PP-OCRv5 Devanagari model
- Document type and strong identifier extraction
- Editable OCR text
- One colour PDF plus OCR summary pages

First OCR use downloads PaddleOCR/ONNX resources. Identification is rules-based and must be reviewed when confidence is low.
