# Smart Archive Tools — App #014

Browser-local ZIP utility for 1 Hub Apps.

## Features

- Create ZIP archives from multiple files
- Add folder contents where browser folder selection is supported
- Remove staged files before ZIP creation
- Open ZIP archives locally
- Browse archive file/folder structure
- Select and extract individual files
- Extract all files
- Chat actions for listing ZIP contents, extracting a named file, and creating a ZIP from an attached file

## Processing

ZIP processing uses the project's existing `jszip` dependency and runs in the browser. No server upload is required by this app.
